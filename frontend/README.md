# Frontend — Fase 10

Frontend profesional para el sistema de salón de belleza.

## Ejecutar

```bash
cp .env.example .env
npm install
npm run dev
```

La API debe estar disponible en `http://localhost:8000/api/v1`. Puede modificarse con `VITE_API_URL`.

## Módulos

- Login JWT y renovación automática del token
- Dashboard con KPIs y gráficos
- Clientes
- Servicios
- Empleados
- Agenda
- Ventas
- Caja
- Inventario
- Compras
- Reportes Excel/PDF
- Diseño responsive y modo oscuro
